## wx_common_pyo3

`rust`终究统一天下?学习一下基于`pyo3`通过`rust`写`python`的扩展

参考资料:

- [pyo3 语言绑定包](https://pyo3.rs/main/getting-started.html)
- [maturin 构建工具](https://www.maturin.rs/project_layout)

打包方式:

```bash
cd ./wx_common_pyo3
uv run maturin develop  --uv # 用uv管理虚拟环境就得这么干
```
